Skoodle AI is a lightweight AI chatbot built using HTML, CSS, and JavaScript, designed with a modern and interactive user interface. The project focuses on creating a simple yet powerful conversational assistant that can respond to user queries, manage chat history, and provide a smooth messaging experience.

The interface is inspired by modern chat platforms like ChatGPT and WhatsApp, where user messages and AI responses appear on opposite sides for better readability.

🚀 Features

💬 Real-time AI chatbot interaction

🧠 Intelligent responses using AI APIs

📜 Chat history system

🎨 Modern glassmorphism UI design

📱 Responsive design for desktop and mobile

🤖 AI avatar and user chat bubbles

🌍 Scalable for multiple languages

🛠️ Technologies Used

HTML5

CSS3

JavaScript

AI API integration (e.g., OpenAI)

🎯 Goal

The goal of this project is to build a simple, customizable AI chatbot interface that developers and businesses can modify for customer support, personal assistants, or productivity tools.
